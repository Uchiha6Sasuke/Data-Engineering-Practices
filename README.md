# Data-Engineering-Practices
Репозиторий, содержащий ноутбуки с проектами по аналитике и исследованию данных

1. Model_Monitoring_Dashboard.ipynb содержит создания дэшборда мониторинга модели машинного обучения.
     1. Загрузка датасета с мошенническими транзакциями по кредитным картам (kaggle);

     2. EDA;

     3. Подготовка данных - разделение на train/test, стандартизация признаков;

     4. Обучение модели - логистическая регрессия с учетом дисбаланса классов;

     5. Оценка метрик;
   
     6. Мониторинг модели - анализ дрейфа признаков и предсказаний;
